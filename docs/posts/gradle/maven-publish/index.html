使用Gradle发布到Maven中央仓库

2020-03-21
阅读时长: 1 分钟

  1. 注册https://issues.sonatype.org/账号

  2. 使用 scoop 安装 gpgscoop install gpg

  3. 使用 gpg 生成密钥对

    gpg --full-generate-key

    导出私钥

    gpg --export-secret-keys -o secring.gpg

    发布到 ubuntu 密钥库

    gpg --keyserver keyserver.ubuntu.com --send-keys ***********
